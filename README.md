# 🌱 AutoPlant - Smart Irrigation System
AutoPlant is an IoT-based automatic plant watering system concept with a web dashboard for monitoring plant conditions and managing irrigation. The web application integrates Firebase for authentication and data management and is designed to work with sensor data from an ESP8266-based system.

## Features
- Real-time moisture monitoring
- Automatic irrigation
- Weather integration
- Water usage history
- Firebase authentication
- Live dashboard

## Technologies Used
- HTML
- CSS
- JavaScript
- Firebase
- ESP8266
- Arduino IDE
