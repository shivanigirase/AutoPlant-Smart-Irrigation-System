📘 About Project

Problem Statement

Traditional plant watering requires manual effort and may lead to overwatering or underwatering. This causes poor plant growth and water wastage.

Project Objective

To build an automatic plant watering system using real-time sensor data to ensure plants get proper water while reducing human effort.

Components Used

  • ESP8266 (NodeMCU)
  • Soil Moisture Sensor
  • DHT11 Sensor
  • Rain Sensor
  • Relay Module
  • Water Pump
  • Firebase
  • Website

System Architecture

Sensors collect data → ESP8266 sends to Firebase → Website displays data → System controls pump automatically.

Working

Sensors monitor moisture, temperature, and humidity. If soil is dry, pump turns ON. If moisture is sufficient or rain is detected, pump turns OFF.

Features

  • Real-time monitoring
  • Automatic irrigation
  • Weather-based control
  • Remote access

Advantages

  • Saves water
  • Reduces manual work
  • Improves plant growth

Limitations

  • Needs internet connection
  • Initial setup cost

Applications

  • Smart farming
  • Greenhouses

Future Scope
